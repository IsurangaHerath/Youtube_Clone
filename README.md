# YouTube Clone

A beginner-friendly YouTube clone built with HTML, CSS, and JavaScript. Perfect for learning web development and building your GitHub portfolio!

## 🎯 Features

- **Homepage with Video Grid** - Displays a responsive grid of video thumbnails with titles and channel names
- **Video Player Page** - Watch videos with a full player interface
- **Search Functionality** - Filter videos by title in real-time
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Like & Subscribe Buttons** - Interactive buttons with visual feedback
- **Comments Section** - View sample comments on each video
- **Related Videos** - Sidebar showing related video suggestions
- **Dark Theme** - Modern YouTube-inspired dark mode design

## 🚀 How to Run the Project

### Option 1: Open Directly in Browser (Easiest)

1. **Locate the files on your computer**
   - Navigate to the folder where you saved these files
   - For example: `D:\Projects\Youtube Clone\`

2. **Double-click `index.html`**
   - This will open the homepage in your default web browser
   - You can also right-click and select "Open with" to choose your browser

3. **Browse the website**
   - Click on any video thumbnail to watch it
   - Use the search bar to filter videos
   - Click "Subscribe" or "Like" to see interactive buttons

### Option 2: Use VS Code Live Server (Recommended for Development)

1. **Install VS Code** (if not already installed)
   - Download from: https://code.visualstudio.com/

2. **Open the project in VS Code**
   - Open VS Code
   - Go to File > Open Folder
   - Select the `youtube-clone` folder

3. **Install Live Server extension**
   - Click the Extensions icon (or press Ctrl+Shift+X)
   - Search for "Live Server"
   - Click Install on the one by Ritwick Dey

4. **Start the server**
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - The website will open in your browser at `http://127.0.0.1:5500/`

### Option 3: Use Python Simple HTTP Server

1. **Open Command Prompt or Terminal**
   - Press Win+R, type `cmd`, press Enter
   - Or use Terminal on Mac

2. **Navigate to the project folder**
   ```bash
   cd D:\Projects\Youtube Clone
   ```

3. **Start the server**
   ```bash
   # For Windows
   python -m http.server 8000
   
   # For Mac/Linux
   python3 -m http.server 8000
   ```

4. **Open in browser**
   - Go to: `http://localhost:8000`
